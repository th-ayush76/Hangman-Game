# Hangman-Game 🎮

A command-line implementation of the classic Hangman word-guessing game built using Python.

## 🛠️ How to run
1. First, make sure you have Python installed on your system.
2. Download or clone this repository to your local machine.
3. Open the Terminal or Command Prompt (CMD) and navigate to the project folder.
4. Run the game by typing the following command:
   `python hangman.py`

## 📝 Rules
- You need to guess the secret word letter by letter.
- Your lives will decrease with every incorrect guess.
- You must guess the correct word before all your lives run out!
